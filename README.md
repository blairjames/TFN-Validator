# TFN-Validator
Australian Tax file number validator

### Usage
Run as a Python script or as a Docker container.

```bash

$ ./tfn_validator.py
$ ./build.sh
$ ./run.sh

```

### Example

```bash

$ ./tfn_validator.py
Enter potential TFN: 149149149
149149149 is a valid TFN.

$ ./tfn_validator.py
Enter potential TFN: 123456789
123456789 is NOT a valid TFN.

```
